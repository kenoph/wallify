# Wallify

Simple python script to put together a nice wallpaper from your Spotify music.

## Requirements

* Linux
* KDE 5 (it's easy to add support for other DEs...)
* DBUS and friends
* A bunch of python packages that I still have to put in a `requirements.txt`
